# VSCode

# shortcuts

```

# cut
ctrl + x

# 同名選択
ctrl + d

# コンソール開く系
ctrl (+ shift) + @

# suggest
ctrl + i / ctrl + space

# show commands
ctrl + shift + p    or  f1

# move line
alt + donw/up

# multi cursor
ctrl + alt + down/up

# insert line at
ctrl (+ shift) + enter

# open file
ctrl + p

# open problem result
ctrl + shift + m

# open history
ctrl + tab
もどる・すすむ

# search/replace in file
ctrl + f/h
# all serach
ctrl + shift + f


```
